Hierchical view of the phases

The user can interact in 3 ways with the visualisation:

  • Zooming: By using a touchpad or mouse the user can zoom in by scrolling or zoom by making a two finger scroll movement on your trackpad.
  • Dragging: By clicking and holding the user can drag the bar from the left to the right (only if already zoomed in)
  • Clicking: It the user clicks on the bottom part of a phase then the user can see the lower levels.
